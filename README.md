# PPAP-Fruits
무인 과일가게 가맹점주 관리자 페이지

## 1. 프로젝트 개요
#### ✨ 무인 과일가게 가맹점주를 위한 DB 설계 및 관리자 페이지를 구현한 프로젝트입니다.
> 2023학년도 1학기 `데이터베이스설계` 과목 팀 프로젝트로 진행한 프로젝트입니다. <br>
> 무인 과일가게 관리를 위한 DB 설계 후 관리자 페이지를 구현했습니다.
